# Spark Vision Backend

Standalone NestJS API for Spark Vision.

## Run

```bash
npm install
npm run dev
```

Default port: `5000`

## Important Scripts

- `npm run build`
- `npm run start:prod`
- `npm run migrate:auth-tracking`
- `npm run optimize:indexes`
- `npm run test:auth-tracking`

## Healthcheck

- `GET /health`
